# Subnautica-quest-text# Subnautica: Текстовый квест

Текстовый квест по мотивам Subnautica. Выживание, исследование пещеры, сбор ресурсов, битва со Жнецом.

## Управление
- Вводите **номер действия** (1, 2, 3...)
- `99` – открыть инвентарь
- `сохранить` / `загрузить` – сохранение и загрузка (только в спасательной капсуле)

## Как играть
1. Потушите пожар в капсуле, возьмите планшет, откройте ящик с припасами.
2. Соберите 2 титана (один в пещере, другой из металлолома).
3. Скрафтите нож.
4. Победите Жнеца (аптечка поможет пережить две атаки).

## Запуск
- **Из Python**: `python game.py`
- **Из готового EXE**: скачайте `SubnauticaQuest.exe` в разделе [Releases](https://github.com/Kamilsamir2/Subnautica-quest-text/releases) и запустите.

## Требования
- Python 3.6+ (только для запуска из исходников)
- Библиотеки: стандартные (`json`, `os`, `random`, `typing`)
